#Zamysł projektu
Nasz program sam wchodzi na stronę internetową z badaniami medycznymi gdzie naukowcy badali jak dorośli z ADHD radzą sobie z kontrolą poznawczą (skupieniem), w porównaniu do osób zdrowych (te osoby były grupą kontrolną). Tabela pokazuje ich czasy reakcji i procent błędów w teście Stroopa.

#Jak to zainstalować?
Prosimy o wpisanie komendy w celu pobrania potrzebnych bibliotek:
pip install -r.requirements.txt

#Jak włączyć nasz program?
Prosimy o wpisanie komendy:
python main.py
